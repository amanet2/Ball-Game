* download JDK 20 from oracle (ew)
* for GCC:
  1. Use mingw gui to install packages(MinGW\libexec\mingw-get\guimain.exe)
  2. Under "Basic Setup" tick "mingw32-base-bin"
  3. Installion -> Apply Changes
  4. Be patient, it takes an uncomfortable amount of time
  5. Edit environment SYSTEM path variable to include "<drive>:\MinGW\bin"
  6. Helpful: http://www.codebind.com/cprogramming/install-mingw-windows-10-gcc/
